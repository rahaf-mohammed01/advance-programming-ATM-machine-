# ATM Machine Simulation System

## Overview
This project is a simulation of an Automated Teller Machine (ATM) system developed as part of an advanced programming course. It allows users to perform basic banking operations through a simple and interactive interface.

The project focuses on applying core programming concepts such as conditionals, loops, and structured logic.

## Features
- User authentication (PIN verification)
- Check account balance
- Withdraw money
- Deposit money
- Simple menu-driven interface

## Technologies Used
- C++   
- Console-based interface

## How It Works
The system simulates real ATM functionality. The user logs in using a PIN, then selects operations from a menu such as checking balance, depositing money, or withdrawing funds.

The program processes user input and updates the account balance accordingly.
